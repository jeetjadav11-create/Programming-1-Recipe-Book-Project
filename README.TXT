PROJECT TITLE: Digital Recipe Book

PURPOSE OF PROJECT: A digital recipe book that mimics the functions of a paper recipe book.

AUTHORS: Gianluca Zambito and Jeet Jadav

HOW TO RUN:
    In BlueJ:
        Create a new object of type RecipeBook, then an object of type of type RecipeBookMenu, and use the object of type RecipeBook as the value for the parameter of the second object.
        
    Outside BlueJ:
        Call the main class, RecipeBookMenu.
